# Zepto_Sales_Analysis
Data Analytics project showcasing Sales of Different Category Products of Zepto using MYSQL. 

# 🛒 Retail Data Analysis Project (MySQL + Excel)

## 📌 Project Overview

This project is based on a real-world retail dataset initially stored in Excel and then imported into MySQL for analysis. The goal of this project is to simulate a real business scenario where raw data is cleaned, transformed, and analyzed to generate meaningful insights.

The dataset contains product-level information such as category, pricing, discounts, stock availability, and quantity.

---

## 📂 Dataset Details

The dataset includes the following columns:

* **Category** – Product category (e.g., Beverages, Dairy, etc.)
* **Name** – Product name
* **MRP** – Maximum Retail Price
* **Discount Percentage**
* **Available Quantity**
* **Discounted Selling Price**
* **Weight (grams)**
* **Out of Stock (Boolean)**
* **Quantity Sold**

## 📊 Data Analysis & Business Questions

The project includes **8 real-world business questions**, such as:

* Identifying top-selling product categories
* Finding products with the highest discounts
* Analyzing price vs. quantity trends
* Detecting out-of-stock patterns
* Revenue estimation based on discounted prices

* Identifying high-demand low-stock products
